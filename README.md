# CardScanner 

A card scanner app for credit cards.

## 	:book: Description :
- This application is designed to allow card scanning using MLKit.
- You can easily scan your cards and then edit them.

  </br>

## 	:gear: Library Versions : 
| Library | Version |
| ----------------- | ----------------- |
| CameraX | 1.3.4 |
| MLKit | 16.0.1 |

</br>
